# Frontend Demos and CSS Styling Assignments

This repository contains fundamental frontend engineering assignments, exploring basic HTML5 structure, semantic elements, CSS styling, flexbox layouts, image float operations, and multi-page static site navigation.

---

## Folder Blueprint

```
Frontend-Demos/
├── Assignment-1/      # Core HTML lists and formatting
├── Assignment-2/      # Home Page structures
├── Assignment-3/      # CSS Integration (Black Goose Bistro layout)
├── Assignment-4/      # Text formatting and image floating layouts
├── Assignment-5/      # Interconnected multi-page city information hub
└── assets/            # Static image assets utilized in the assignments
```

---

## Assignment Breakdown

### Assignment 1: Core HTML Structures
Focuses on semantic tags, basic nesting, and list systems.
* `NestedUnorderedList.html`: Demonstrates bullet nested sub-lists isolating categories like fruits, vegetables, and cereals.
* `webpage.html`: Displays basic text blocks and element structures.

### Assignment 2: Standard Home Page Layout
* `HomePage.html`: A structural exercise mapping core divisions, headings, and paragraph blocks.

### Assignment 3: CSS Integration (Black Goose Bistro)
Features custom class selectors, inline rules, and external style sheets.
* `index.html`: Web page for the Black Goose Bistro restaurant. Features images, catering services, schedules, and custom font weights.
* `styles.css`: External style sheets detailing margins, centered text alignments, rules, and container properties.

### Assignment 4: Float Layouts and Wrapping
Focuses on advanced block element alignments and wrapping text around floated boundaries.
* `layout.html`: Illustrates image alignments (floating sunrise illustrations right and walkers left) wrapped around standard Lorem Ipsum paragraphs.
* `index.html` & `styles.css`: Formatting exercises utilizing floats.

### Assignment 5: Cities Navigation Hub
A multi-page navigation platform linking directories and localized image files.
* `Cities.html`: Index hub mapping paths to Bangalore, Chennai, and Hyderabad.
* `hyderabad.html`, `chennai.html`, `bangalore.html`: Specialized destination profiles displaying landmarks, locations, and travel indices.
* `styles.css`: Styling configurations for text container cards, lists, borders, and margins.

## Viewing the Demos Locally

Since these assignments are composed of standard, client-side static HTML and CSS:

### Method A: Browser Launch
1. Navigate to any assignment folder.
2. Double-click the `.html` file (e.g., `Cities.html`) to open it directly in Google Chrome, Microsoft Edge, or Mozilla Firefox.

### Method B: Live Server (Recommended)
1. Open the repository folder inside VS Code.
2. Install the **Live Server** extension.
3. Click **Go Live** on the bottom right of the IDE status bar to view changes dynamically.
